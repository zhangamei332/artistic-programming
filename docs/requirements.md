# 需求规格说明书

## 1. 项目概述

**产品名称**：艺术编程交互平台（暂定）

**一句话描述**：面向非编程用户的网页端 AI 艺术编程工具，通过自然语言或图片输入生成 Three.js / p5.js 艺术代码，并以可视化节点系统呈现。

**目标用户**：不懂代码但想创作数字艺术的用户。

**产品愿景**：做成可公开访问的 SaaS 产品。

---

## 2. 功能需求

### 2.1 初始化生成

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 文生代码 | 用户输入自然语言描述，AI 生成 Three.js/p5.js 代码 | P0 |
| 图生代码 | 用户上传图片，AI 分析后生成对应风格的艺术代码 | P1 |
| 模型选择 | 用户可选择 Gemini / GPT / DeepSeek 等模型 | P1 |
| 代码类型自动判断 | AI 自动判断生成 Three.js 还是 p5.js | P0 |

### 2.2 实时预览

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 代码运行预览 | 右侧 50% 区域实时渲染 Three.js/p5.js 效果 | P0 |
| 自动刷新 | 代码变化时预览自动更新 | P0 |
| 错误提示 | 代码运行出错时显示错误信息 | P1 |

### 2.3 节点系统

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 自动解析 | 代码生成后自动解析为节点模块 | P0 |
| 节点画布 | 左侧 50% 区域，可拖拽、连线、删除节点 | P0 |
| 参数调整节点 | 控制数值型变量 | P0 |
| 色彩控制节点 | 控制颜色/调色板参数 | P0 |
| 按键交互节点 | 控制键盘事件绑定 | P1 |
| 图像映射节点 | 控制纹理/图片映射 | P1 |
| 摄像机交互节点 | 控制 3D 相机/鼠标交互 | P1 |
| 视频输出节点 | 控制录制/导出 | P2 |
| 硬件接入节点 | 控制外部设备接入 | P2 |
| 节点连线 | 数据流方式的节点连接 | P1 |
| 节点文生微调 | 选中节点，用文字对局部代码微调 | P1 |

### 2.4 代码编辑

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 代码编辑器 | Monaco Editor，语法高亮、自动补全 | P0 |
| 模式切换 | 节点模式 ↔ 代码编辑模式一键切换 | P0 |
| 双向同步 | 代码修改后节点更新，节点修改后代码更新 | P0 |

### 2.5 用户系统

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 注册/登录 | 邮箱注册 + 第三方登录 | P1 |
| 项目保存 | 保存项目到云端，随时继续编辑 | P1 |
| 项目分享 | 生成分享链接，他人可查看 | P2 |

### 2.6 积分与会员

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 积分消耗 | 每次 AI 生成消耗积分 | P1 |
| 积分定价 | 以 API 价格 1.5 倍为基准 | P1 |
| 月度会员 | 每月固定积分，多档可选 | P2 |
| 新用户体验积分 | 注册赠送 | P1 |

---

## 3. 非功能需求

| 需求 | 描述 |
|------|------|
| 性能 | 代码生成后 30s 内完成节点解析和预览渲染 |
| 安全 | API Key 仅存储在后端，不暴露前端 |
| 可用性 | 新手 5 分钟内可完成第一次生成 |
| 兼容性 | 支持 Chrome / Edge / Safari 最新版 |
| 响应式 | 桌面端优先，移动端暂不考虑 |

---

## 4. 约束条件

- 首期仅中文
- 后端语言：Node.js (TypeScript)
- 前端框架：React (TypeScript)
- 数据库：PostgreSQL
- 部署：待定
