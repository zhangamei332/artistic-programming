## 第 1 阶段：核心 MVP — 文生代码

**时间**：约 2-3 周  
**目标**：打通「输入文字 → AI 生成代码 → 实时预览」全链路

- [ ] 后端：接入 DeepSeek API
- [ ] 后端：文生代码 API（`POST /api/generate/text`）
- [ ] 前端：底部输入面板（文字描述 + 模型选择）
- [ ] 前端：Monaco Editor 代码编辑器
- [ ] 前端：iframe sandbox 预览窗口
- [ ] 前后端联调，完整链路验证

**验收标准**：输入「一个旋转的彩色立方体」→ 右侧预览看到旋转立方体

---

## 第 2 阶段：代码解析 + 节点系统

**时间**：约 2-3 周  
**目标**：代码自动解析为可视化节点，节点可交互、可连线

- [ ] 后端：代码解析引擎（AST 分析 → 提取参数/颜色/交互模块）
- [ ] 前端：React Flow 节点画布（拖拽、缩放、连线）
- [ ] 前端：7 种节点类型 UI 组件
- [ ] 前端：节点参数面板（修改 → 代码更新 → 预览刷新）
- [ ] 前端：节点 ↔ 代码双向同步

**验收标准**：生成代码后左侧自动出现节点，修改节点参数后预览实时变化

---

## 第 3 阶段：图生代码

**时间**：约 1 周  
**目标**：支持上传图片生成艺术代码

- [ ] 后端：图片上传 + AI Vision API 分析
- [ ] 后端：图生代码 API（`POST /api/generate/image`）
- [ ] 前端：图片上传组件 + 预览

**验收标准**：上传一张图片 → 生成模仿该图片风格的艺术代码

---

## 第 4 阶段：用户系统

**时间**：约 1-2 周  
**目标**：注册登录、项目云端保存

- [ ] 后端：用户注册/登录 API（JWT + bcrypt）
- [ ] 后端：项目 CRUD API
- [ ] 前端：登录/注册页面
- [ ] 前端：项目列表、保存、加载

**验收标准**：注册 → 登录 → 创建项目 → 保存 → 退出 → 重新登录 → 加载项目
