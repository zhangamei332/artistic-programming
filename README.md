# 艺术编程交互平台

面向非编程用户的网页端 AI 艺术编程工具。输入文字或上传图片，AI 自动生成 Three.js / p5.js 艺术代码，并通过可视化节点系统轻松调整效果。

## 技术栈

- **前端**：React + TypeScript + Vite
- **后端**：Node.js + Express + TypeScript
- **数据库**：PostgreSQL + Prisma
- **AI**：Gemini / GPT / DeepSeek

## 快速开始

### 前端
```bash
cd frontend
npm install
npm run dev
```

### 后端
```bash
cd backend
npm install
npm run dev
```
